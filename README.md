# Парсер статей с Pedsovet.org

Парсер для сбора информации о статьях с образовательного портала [Pedsovet.org](https://pedsovet.org/). Извлекает заголовки и ссылки на статьи из карточек на главной странице.

## Установка и запуск

1. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

2. **Запустите парсер:**
```bash
python parse_pedagogy.py
```

## Результат

Программа создает файл `articles.json` со структурой:
```json
[
  {
    "title": "Название статьи",
    "link": "https://pedsovet.org/article/ссылка-на-статью"
  }
]
```

Также выводит список найденных статей в консоль.

## Зависимости

- beautifulsoup4
- requests
- lxml

## Особенности

- Точно определяет карточки статей
- Обрабатывает относительные ссылки
- Исключает дубликаты
- Сохраняет данные в JSON-формате
